このページにいる人はUnityをダウンロードした人...だよね？
まぁ、私はその体で話を進めるしかないんだけどねｗ
もしUnityをダウンロードしてないよ～って人は、前のページ（Unityをダウンロードしよう！）を参考にしてみてね


さて、前回はUnityをダウンロードしただけだから、一緒にUnityの画面の見方を勉強しよ～


まずは前回ダウンロードしたUnityを開こっか
左のメニューからプロジェクトを押そ～



まだ何もないね。この部分は大雑把に言うと、君が作ったゲームを管理する画面だね。
"プロジェクト ＝ 自分で作成しているゲーム"って思ってくれて構わないよ。



それじゃあ右上の"+新しいプロジェクトを作成"を押そう。



なんかいっぱい出てきたね。この画面は大きく分けるとテンプレート選択とファイル作成を設定する画面だね
※伏せているのは身バレ防止



１個ずつ見てみよっか


■テンプレート選択
テンプレート選択...正直詳しくは分かってないんだけどね。
感覚的には"どんなゲームを作りたいですか～"ってやつかな


例えば、2Dゲームを作りたいって思ったら"Universal 2D"のテンプレートを選択するって感じだね



他にも、サンプルとか、学習用のテンプレートとかいろいろなやつがあるね
自分がやりたい・作りたいやつに合わせてテンプレートを選んでいこ～



■ファイル作成
これはファイルを"どんな名前で、どこに保存しますか"って言うやつだね


まずはファイルの名前（プロジェクト名）を決めていこっか
名前の決め方は"プロジェクト名"ってところで決めれるよ
分かれば何でもいいんだけど、そうだなぁ～..."my_first_game"って名前にしようかな



次にどこに保存するかを決めていこっか
と言っても、自分の好きな場所に保存したら良いんだけどねｗ


私はデスクトップ上に新しく"make_game"って名前のフォルダを作成してその中に保存しようかな



それじゃ保存場所を設定していくよ～
まずは保存場所のとこをクリックしてみよ～



そしたら、なんか選択するやつが表示されたと思う
※伏せているのは、まぁ...あんまり見られたくないからｗ（フォルダのネーミングセンスとか）



私はさっきデスクトップ上に作成した"make_game"に保存したいから、
左のデスクトップを押して"make_game"を選択したら"フォルダーを選択"をクリック



保存場所の最後のところが"\make_game"になってたらOK




テンプレート選択とファイル作成ができたら、"+プロジェクトを作成"を押そっか
※なんか画像が間違ってるけど、私は"Universal 2D"を選んでるよ(さっき間違いに気づいた。あと画像を修正するの面倒なの)



なんかこんなの出てきたかな



これはUnityを立ち上げてる準備をしてるんだよ
時間がかかるから焦らず気長に待とっか～


Unityが立ち上がったらこんな画面が出てきたかな



基本的にこの画面でゲームを作っていくよ
そして、ここからが本題、Unityの画面構成の話だよ


Unityの画面は主に

  • Scene View（シーンビュー）・Game View（ゲームビュー）
  • Hierarchy（ヒエラルキー）
  • Inspector（インスペクター）
  • Project（プロジェクト）・Console（コンソール）
で構成されてるよ



一個ずつ見ていこう


■Scene View（シーンビュー）・Game View（ゲームビュー）


-Scene View（シーンビュー）

ここはオブジェクトの配置を視覚的に決めるとこだね。



例えば、箱オジェクトがあったとして（箱オブジェクトの出し方は後から記載）



箱を動かしたり






回転したり





伸縮したりとか







他にも左のとこにあるツールを使っていろんなことができるよ～



-Gcene View（ゲームビュー）

Gcene View（ゲームビュー）は実際にゲームをするとき、どんな画面なのかを示してくれるビューだよ。


screen view（スクリーンビュー）からGcene View（ゲームビュー）に切り替えるには、
上にあるタブを"Scene"から"Game"のタブに切り替えるだけ



さっき動かしたりした白い箱がちゃんと表示されてるね。


あっ、あと画面の比率もこのゲームビューで変えれるよ。


Gameタブの下くらいに"〇〇〇 Aspect"ってのがあると思う（〇〇〇は比率とかfree）
ここを押せば画面の大きさを変えれるよ。
例えば、"free Aspect"にしたら、ユーザーの画面の大きさによって画面の比率が変わったり、




"16:9 Aspect"なら、
横が16,縦が9の比率で画面の大きさが調節されるよ




実際にゲームを動かしてみたいときは、
画面上部にあるplayボタン（▶）を押せば実際にゲームが動くよ



一応、ゲームが動いている状態の時でもオブジェクトの配置や設定は変えれるよ。
でも、playをやめたら、変更した設定は全部リセットされて、play前の状態に戻る
だから...まぁ...気を付けてね


playを止めたいときはまたplayボタン（今度は■）を押せば止まるよ。



ついでに...
一時停止（⏸）はゲームを一時停止、ゲームの処理を止めるよ
ステップ(⏵❘)は1フレームだけゲーム進めるよ。デバッグに便利


■Hierarchy（ヒエラルキー）

Hierarchy（ヒエラルキー）はゲームの画面に置いてあるオブジェクトを表示するところだよ。
